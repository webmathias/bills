This project is a node/web with angularjs to you control your daily bills to pay and receive.

Configure Environment
  PORT (Default: 3000)
  DB_USER (Default: '')
  DB_PASS  (Default: '')
  DB_HOST (Default: 'localhost')
  DB_PORT (Default: '27017')
  DB_NAME (Default: 'contas')
  
Run:

node node/index.js
